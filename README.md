### Description

一套平衡的Boss设置, Boss击杀天使两次可打响指.


### Boss隐藏技能: 响指!

- 条件: 击杀天使两次  
- 触发: 按Z键 发送终极技能状态  
- 效果: 随机掐死一个敌人
