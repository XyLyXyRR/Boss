# 猪锤Boss

- [熔火工坊](https://owmod.net/226)

> - Name: 猪锤Boss
> - Author: `XyLyXyRR`
> - Code: `K099S`

一套平衡的Boss设置, Boss击杀天使两次可打响指. 

### Description

    源氏Bug刀: 第二次点击左键触发, 普通攻击会自动砍一刀, 如果无法做任何动作, 使用`爬墙/近战/技能`可恢复.    Boss击杀天使两次可打一次响指, 随机掐死一个, 按Z键发送终极技能状态触发.

## Boss隐藏技能: 响指!

- 条件: 击杀天使两次  
- 触发: 按Z键 发送终极技能状态  
- 效果: 随机掐死一个敌人
