### Description

    最佳后死, 队1CD较短  无最佳连续三局最佳当Boss.  两个输出可以打死Boss. 需要配合, 不然就是切菜.    Boss击杀天使两次可打一次响指, 随机掐死一个, 按Z键发送终极技能状态触发.

> Boss隐藏技能: 响指!
> - 条件: 击杀天使两次  
> - 触发: 按Z键 发送终极技能状态  
> - 效果: 随机掐死一个敌人
